# Simple Discord Music Bot

**Disclaimer: This project is for educational purposes. Please respect copyright laws and Discord's Terms of Service.**

# Prerequisites

- Python 3.12+

- FFmpeg: Must be installed and added to your system's PATH.

- Deno or Node.js: Recommended for yt-dlp to handle YouTube's JavaScript signatures.

## Getting Started:

- **Clone and Setup:**

```bash
uv sync
```

- **Configuration:** Create a `.env` file in the root directory:

```bash
BOT_TOKEN="your_bot_token_here"
```

- **Run:**

```bash
uv run main.py
```
