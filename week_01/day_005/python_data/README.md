# Python Data Manipulation

This code is purely is build for learning core python data libraries.
Hopefully, it will cover `numpy`, `pandas`, `matplotlib`, `seaborn` etc.

## Progression

- [ ] **Compute** (`NumPy`): Fast numerical foundations.
- [ ] **Wrangle** (`Pandas`): Data cleaning and manipulation.
- [ ] **Blueprint** (`Matplotlib`): Basic plot architecture.
- [ ] **Story** (`Seaborn`): Statistical visual intelligence.
- [ ] **Pipeline** (`Integration`): Streamlined data transformation and machine learning workflows.

**Goal:** Converting raw signals into actionable insights.
